事实边界

Markdown MemoryItem 是事实源；raw transcript/resource 是证据旁路；index/vector/runtime/review 都是派生或复核层。算法栈解释候选排序；召回不等于注入，注入内容由 ContextFirewall 和 ContextPack 处理。
