install

doctor
memory adapter doctor codex --format json

One local brain pool.

AMH 把原始证据、长期记忆、索引和上下文包分层存放。Agent 看到的是经过裁剪的 context pack，不是整仓 transcript。

~/.agent-memory-hub/
  • items/长期 MemoryItem
  • sources/conversation evidence
  • indexes/FTS / vector / refs_graph
  • resources/项目文件、知识源、外部资料
MemoryItem
decision

非显然技术决策，必须写理由和改回去的代价。

fact

客观限制、配置、数字、API 行为，必须写来源和有效期。

artifact

PR、文档、文件、工具，必须写产出物路径和用途。

handoff

承接工作时用 brief 优先，只读取必要的 1 到 3 条正文。

Recall with governance.

召回不是把所有记忆直接注入 prompt，而是先过滤项目、类型、时间、敏感度，再把 locator、overview、detail hint 组成可控上下文包。

query_signal
1. detect

识别当前任务关键词、cwd、adapter、文件边界。

2. retrieve

先拿压缩视图，避免开局 bulk-read 吃光上下文。

3. verify

关键判断再 bounded read 正文，和当前工具证据交叉验证。

memory brief
memory brief --project agent-memory-hub --limit 8
memory read mem_xxx --view detail --head 2000
